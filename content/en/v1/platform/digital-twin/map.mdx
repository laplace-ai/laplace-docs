# Interactive Map

The interactive map is the primary visualization of the Digital Twin module. It displays your logistics network as a geographic view with facility markers, route connections, and interactive details.

## Map controls

### Mouse and gesture controls

| Action | How |
|--------|-----|
| **Zoom in** | Scroll wheel up, or pinch outward on trackpad |
| **Zoom out** | Scroll wheel down, or pinch inward on trackpad |
| **Pan** | Click and drag on the map |
| **Rotate** | Right-click and drag |
| **Reset rotation** | Double-click on the compass (if visible) |

### Keyboard shortcuts

| Key | Action |
|-----|--------|
| `+` / `=` | Zoom in |
| `-` | Zoom out |
| Arrow keys | Pan the map |

## Map layers

The map shows information in distinct layers:

### 1. Base map

The base layer provides geographic context — roads, cities, water bodies, and terrain. It automatically switches between light and dark variants based on your theme setting.

### 2. Route lines

Route connections between units are displayed as lines on the map. Each line connects two facilities and represents a freight route in your network.

Route lines have two visual states:

- **Default**: Thin, semi-transparent lines showing the full network
- **Highlighted**: When a unit is selected, routes connected to that unit become more prominent while others fade

If the same pair of facilities has multiple routes (e.g., different vehicle types), only one line is drawn between them to keep the map clean.

### 3. Unit markers

Unit markers appear at each facility's location using a blue color. Each marker responds to:

- **Hover**: Shows the unit code as a tooltip
- **Click**: Opens the unit detail panel and highlights connected routes

## Unit detail panel

When you click a unit marker, a panel slides in from the right side of the map with the following information:

### Header section

- **Unit code** (e.g., "ATI SPA") displayed in blue
- **Active status** indicator — a green dot for active units, a gray dot for inactive ones
- **Unit name** (e.g., "Filial Sao Paulo")
- **Location** showing city and state
- **Close button** (X) in the top-right corner

### Overview section

Key information about the unit:

- **Type**: The facility type (hub, branch, distribution center, etc.)
- **Status**: Active or Inactive

### Services section

Currently displays a "Coming soon" placeholder. In future releases, this section will show:

- Loss prediction summary for the unit
- Risk distribution charts
- Operational insights and alerts

### Footer

A button labeled **"View Detailed Unit"** that navigates to the dedicated unit detail page with full operational data.

## Immersive mode

The Digital Twin map supports an **immersive mode** that maximizes the map area by hiding the top and bottom bars. When immersive mode is active:

- The map extends to fill the full available height
- The unit detail panel adjusts accordingly
- The sidebar remains visible for navigation

Immersive mode is automatically exited when you navigate away from the Digital Twin page.

## Responsive behavior

The map adapts to layout changes:

- **Sidebar collapse/expand**: The map automatically resizes when the sidebar state changes, filling the available space without gaps.
- **Window resize**: The map handles window size changes automatically.
- **Panel open/close**: The unit detail panel overlays the map, so the map content shifts behind the panel.

## Map configuration

The map is configured to show a broad view of Brazil by default:

| Setting | Value |
|---------|-------|
| Initial view | Geographic center of Brazil |
| Initial zoom | Shows the full national network |
| Marker color | Blue (Laplace accent color) |
