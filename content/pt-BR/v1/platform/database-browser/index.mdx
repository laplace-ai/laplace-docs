# Explorador de Dados

O Explorador de Dados fornece acesso direto as tabelas de dados que alimentam a Plataforma Laplace. E util para validacao de dados, investigacao de embarques especificos e compreensao das informacoes subjacentes.

Navegue ate **Database** na barra lateral.

## Layout da interface

O Explorador de Dados utiliza um **layout de dois paineis**:

### Painel esquerdo — Lista de tabelas

Uma barra lateral (dentro da area de conteudo principal) lista todas as tabelas disponiveis. Clique em um nome de tabela para carregar seus dados na grade principal.

Tabelas disponiveis:

| Tabela | Descricao |
|-------|-------------|
| **CTRCs** | Registros de embarque — os dados operacionais principais |
| **Predicoes de Perda** | Resultados de predicao incluindo probabilidade, nivel de risco e confianca |
| **Unidades** | Dados de instalacoes com codigos, nomes, localizacoes e coordenadas geograficas |
| **Links de Rede** | Conexoes de rotas entre instalacoes com tipo de veiculo, classificacao e precificacao |

### Painel direito — Grade de dados

A area principal exibe uma tabela rolavel com os dados da tabela selecionada.

## Tabelas disponiveis

### CTRCs

A tabela de CTRCs contem dados de embarque. Este e o conjunto de dados operacionais principal.

| Coluna | Descricao |
|--------|-------------|
| **CTRC** | Numero do documento de embarque |
| **Criado** | Data de criacao do CTRC |
| **Localizacao Atual** | Localizacao atual da carga |
| **Unidade de Origem** | Unidade emissora (instalacao de origem) |
| **Unidade de Destino** | Unidade receptora (instalacao de destino) |
| **Remetente** | Nome do cliente remetente |
| **Destinatario** | Nome do cliente destinatario |
| **Cidade de Origem** | Cidade de origem |
| **Cidade de Entrega** | Cidade de entrega |
| **Estado de Entrega** | Estado de entrega (ex.: SP, PR, RJ) |
| **Valor do Frete** | Valor do frete em BRL |
| **Valor da Carga** | Valor da carga em BRL |
| **Peso (kg)** | Peso real da carga em quilogramas |
| **Volumes** | Quantidade de volumes da carga |
| **Mercadoria** | Tipo de mercadoria |

### Predicoes de Perda

A tabela de predicoes contem os resultados de pontuacao dos CTRCs.

| Coluna | Descricao |
|--------|-------------|
| **CTRC** | O numero do embarque pontuado |
| **Probabilidade** | Probabilidade de perda (0% a 100%) |
| **Nivel de Risco** | ALTO, MEDIO ou BAIXO |
| **Confianca** | O quao certo o modelo esta sobre a predicao |
| **Predicao** | Se o modelo preve perda ou nao perda |
| **Versao do Modelo** | Qual versao do modelo produziu esta predicao |

### Unidades

A tabela de unidades contem dados de todas as instalacoes da rede logistica.

| Coluna | Descricao |
|--------|-------------|
| **Codigo** | Codigo unico da instalacao (ex.: "ATI SPA") |
| **Tipo** | Tipo de instalacao (hub, filial, centro de distribuicao, etc.) |
| **Nome** | Nome da instalacao |
| **Cidade** | Cidade da localizacao |
| **Estado** | Sigla do estado |
| **Ativo** | Se a unidade esta atualmente ativa |
| **Coordenadas** | Latitude e longitude geograficas (usadas para posicionamento no mapa) |

### Links de Rede

A tabela de links de rede mostra conexoes de rotas entre instalacoes.

| Coluna | Descricao |
|--------|-------------|
| **Origem** | Codigo da unidade de origem |
| **Destino** | Codigo da unidade de destino |
| **Tipo de Veiculo** | Tipo de veiculo nesta rota |
| **Classificacao** | Classificacao da rota |
| **Preco (BRL)** | Precificacao da rota em BRL |

## Pesquisa

Cada visualizacao de tabela inclui uma barra de pesquisa. Digite qualquer valor e a tabela filtra instantaneamente para linhas que contenham aquele texto em qualquer coluna.

A pesquisa e:

- **Sem distincao de maiusculas/minusculas**: "sao paulo" encontra "Sao Paulo"
- **Correspondencia por substring**: "SPA" encontra "ATI SPA" e "SPA001"
- **Entre colunas**: Pesquisa em todas as colunas simultaneamente

A contagem de resultados atualiza em tempo real, mostrando o numero de linhas filtradas vs. total (ex.: "23 de 150 linhas").

## Carregamento de dados

O Explorador de Dados mostra **ate 100 linhas** por tabela. Ele e projetado como uma ferramenta de consulta rapida, nao como uma interface completa de exportacao de dados.

Quando voce alterna entre tabelas:

- A consulta de pesquisa e limpa
- Um indicador de carregamento aparece enquanto os dados sao buscados
- Os dados da nova tabela substituem a visualizacao anterior

<Callout type="info">
O Explorador de Dados mostra um snapshot das primeiras 100 linhas de cada tabela. Para acesso completo aos dados, use a funcionalidade de exportacao CSV na tabela de Predicao de Perda.
</Callout>

## Casos de uso

O Explorador de Dados e particularmente util para:

| Caso de Uso | Como |
|----------|-----|
| **Validacao de dados** | Verifique que os dados foram carregados corretamente conferindo linhas individualmente |
| **Investigacao** | Consulte um CTRC especifico para ver seus detalhes e predicao |
| **Consulta de unidades** | Encontre a localizacao, tipo ou status de uma unidade |
| **Analise de rotas** | Navegue pelos links de rede para entender conexoes de rotas e precos |
